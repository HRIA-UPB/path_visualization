Task: Visual Grounding

Given a pathology image and a natural language expression describing a pathological feature, the model predicts the bounding box coordinates [x_min, y_min, x_max, y_max] of the region that matches the description.

Model & Method

Qwen 2.5 VL 7B vision-language model fine-tuned using Group Relative Policy Optimization (GRPO), a reinforcement learning algorithm that computes advantages by comparing rewards within sample groups.

RefPath Dataset

27,610 histopathological images with 33,500 expert-verified annotations. Features cellular structures, tumor infiltration patterns, inflammatory responses, and tissue architecture abnormalities.

Reward Structure

Format reward (10%): validates output structure. Accuracy reward (90%): Intersection over Union (IoU) between predicted and ground truth bounding boxes.
